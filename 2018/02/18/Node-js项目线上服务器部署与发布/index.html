使用nginx 80端口

停止apache 服务
sudo service apache2 stop


配置

  1. cd /etc/nginx/
  2. cd conf.d
  3. sudo vi shoukailiang-cn-8081.conf
    写入 4.sudo nginx -t 测试是否成功
  4. sudo nginx -s reload 重启 nginx

之后输入ip地址 就可以看到了 把8081 的服务都导向到Nginx的80端口
